Implement (asm ...) as a special form in the cross-compiler (compile.h). When the compiler encounters (asm "instr1" "instr2" ...), emit each string argument as a raw COR24 assembly line in the output. Steps: (1) Add sym_asm to eval.h and intern it in eval_init().
