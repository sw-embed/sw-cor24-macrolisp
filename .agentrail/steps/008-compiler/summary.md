Implemented Lisp-to-COR24 native code compiler (compile.h, 450 lines). Stack-based codegen: each expression leaves result in r0. Supports fixnums, symbols, quote, if, begin, define, lambda (no closures), inlined +/-/* and </=, runtime calls for cons/car/cdr, predicates, print, and general function calls. Emits valid COR24 assembly with proper calling conventions. Test compiles factorial, double, and arithmetic - all produce correct assembly. Also added REPL and UART input support.
