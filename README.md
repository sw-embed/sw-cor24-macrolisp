# tml24c

Tiny Macro Lisp compiled to the COR24 toolchain.

A minimal Lisp system that compiles to COR24 assembly, using:
- [tc24r](https://github.com/sw-vibe-coding/tc24r) -- Tiny COR24 compiler in Rust
- [cor24-rs](https://github.com/sw-embed/cor24-rs) -- COR24 assembler and emulator

## Status
